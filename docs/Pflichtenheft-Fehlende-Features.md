# Pflichtenheft: Fehlende Features — Chat App vs Slack/Hub

> **Stand:** 14.04.2026
> **Referenz:** Slack UI, Waldkraft Hub Chat-Modul

---

## 1. Profil-Menue (Slack-Stil)

Klick auf eigenes Profilbild/Namen in der Sidebar oeffnet ein Menue mit:

- [ ] **Avatar + Name + Praesenz-Status** (Aktiv/Abwesend mit gruener/grauer Anzeige)
- [ ] **"Deinen Status aktualisieren"** — Klick oeffnet Status-Dialog (Emoji + Text + Ablauf)
- [ ] **"Setze dich auf abwesend/aktiv"** — Toggle Praesenz sofort
- [ ] **"Benachrichtigungen pausieren"** — Submenue: 30 Min / 1 Std / 2 Std / Bis morgen / Benutzerdefiniert
- [ ] **Trennlinie**
- [ ] **"Profil"** — Oeffnet eigene Profilansicht (vollstaendige Bearbeitung)
- [ ] **"Persoenliche Einstellungen" (Cmd+,)** — Oeffnet Einstellungsseite
- [ ] **Trennlinie**
- [ ] **"Downloads" (Cmd+Shift+J)** — Zeigt heruntergeladene Dateien
- [ ] **Trennlinie**
- [ ] **"Abmelden"** — Logout

---

## 2. Sidebar Navigation Rail (linke Icon-Leiste)

Vertikale Icon-Leiste am linken Rand (wie Slack/Hub):

- [ ] **Home** — Startseite mit Uebersicht
- [ ] **DMs** — Direktnachrichten-Ansicht
- [ ] **Kanaele** — Channel-Browser
- [ ] **Aktivitaet** — Erwaehnungen und Reaktionen (mit Badge-Zaehler)
- [ ] **Threads** — Alle Threads an denen man beteiligt ist
- [ ] **Spaeter** — Gespeicherte/gemerkte Nachrichten
- [ ] **Mehr** — Erweitertes Menue (Dateien, Personen, etc.)
- [ ] **Compose-Button** (+) — Neue Nachricht/Channel erstellen

---

## 3. Channel-Header Toolbar (Icons)

Rechts im Channel-Header eine vollstaendige Icon-Leiste:

- [ ] **Stern/Favorit** — Channel als Favorit markieren
- [ ] **Mitglieder-Avatare + Zaehler** (+26) — Klick oeffnet Mitglieder-Panel
- [ ] **Huddle/Telefon-Icon** — Audio-Huddle starten
- [ ] **Video-Icon** — Video-Call starten (Platzhalter)
- [ ] **Pin-Icon** — Angepinnte Nachrichten Panel oeffnen
- [ ] **Bookmark/Link-Icon** — Channel-Bookmarks anzeigen
- [ ] **Personen-Icon** — Mitglieder-Panel oeffnen
- [ ] **Suche-Icon** — In-Channel-Suche oeffnen
- [ ] **Einstellungen-Icon** — Channel-Einstellungen Dialog

---

## 4. Rechte Seitenleisten (Panels)

- [ ] **Mitglieder-Panel** — Liste aller Channel-Mitglieder mit Praesenz, Rolle, Suchfeld
- [ ] **Angepinnte Nachrichten Panel** — Alle gepinnten Nachrichten chronologisch
- [ ] **Gespeicherte Nachrichten Panel** — Alle gespeicherten Nachrichten (In Progress / Erledigt / Archiviert)
- [ ] **In-Channel-Suche Panel** — Suche nur innerhalb des aktuellen Channels
- [ ] **Channel-Bookmarks Panel** — Verwaltung der Bookmark-Leiste

---

## 5. Nachrichten-Erweiterungen

- [ ] **Nachricht weiterleiten** — An anderen Channel/DM weiterleiten mit Vorschau
- [ ] **Nachrichten-Link kopieren** — Deep-Link zur Nachricht
- [ ] **Als ungelesen markieren** — Ab dieser Nachricht als ungelesen setzen
- [ ] **Erinnern** — Erinnerung setzen (30 Min, 1 Std, 3 Std, Morgen, Naechste Woche)
- [ ] **Bild-Lightbox** — Bilder in Vollansicht oeffnen (Klick auf Bild-Vorschau)
- [ ] **Datei Drag & Drop Zone** — Visuelles Feedback beim Ziehen von Dateien ueber den Chat
- [ ] **Nachrichtenbearbeitungs-Historie** — Verlauf der Bearbeitungen einsehbar
- [ ] **Lesebestaetigungen** — Wer hat die Nachricht gelesen (optional)
- [ ] **Datum-Trennlinien** — "Donnerstag, 26. Maerz" / "Gestern" / "Heute" zwischen Nachrichten

---

## 6. Nachrichtenkomposition (Input-Bereich)

- [ ] **Mikrofon-Icon** — Sprachnachricht aufnehmen
- [ ] **Smiley-Icon** — Emoji-Picker oeffnen (rechts im Input)
- [ ] **@-Icon** — Mention-Picker oeffnen
- [ ] **Drei-Punkte-Menue (...)** — Weitere Optionen (Formatierung, Poll, Snippet)
- [ ] **Pfeil-hoch/runter bei Send-Button** — Dropdown: Nachricht planen
- [ ] **Datei-Upload Drag & Drop** mit visueller Drop-Zone (ganzer Chat-Bereich)

---

## 7. Channel-Verwaltung

- [ ] **Channel-Browser** — Alle oeffentlichen Channels durchsuchen und beitreten
- [ ] **Channel-Einstellungen Dialog** — Vollstaendige Konfiguration (Name, Topic, Beschreibung, Posting-Berechtigungen, Benachrichtigungen)
- [ ] **Einladungslink-Dialog** — Link generieren zum Einladen
- [ ] **Sortierung** — Channels sortieren: Aktiv / A-Z (Toggle in der Sidebar)

---

## 8. Ansichten (Views)

- [ ] **Threads-Ansicht** — Eigene Seite mit allen Threads an denen man beteiligt ist
- [ ] **Aktivitaet-Ansicht** — Alle @-Erwaehnungen und Reaktionen auf eigene Nachrichten
- [ ] **Spaeter/Saved-Ansicht** — Alle gespeicherten Nachrichten mit Status-Tabs
- [ ] **Dateien-Ansicht** — Alle geteilten Dateien ueber alle Channels
- [ ] **Personen-Verzeichnis** — Alle Workspace-Mitglieder mit Suche und Filter
- [ ] **Ungelesen-Ansicht** — Alle Channels mit ungelesenen Nachrichten in einer Liste

---

## 9. Persoenliche Einstellungen (Settings-Seite)

- [ ] **Benachrichtigungen** — Desktop-/Mobile-Einstellungen, Sound, DND-Zeitplan
- [ ] **Darstellung** — Theme (Hell/Dunkel), kompaktes Layout, Schriftgroesse
- [ ] **Sprache & Region** — Sprache, Zeitzone, Datumsformat
- [ ] **Erreichbarkeit** — Barrierefreiheits-Einstellungen
- [ ] **Tastenkuerzel** — Uebersicht aller Shortcuts

---

## 10. Sonstige fehlende Features

- [ ] **Datum-Separatoren in Nachrichtenliste** — "Heute", "Gestern", "Montag, 7. April"
- [ ] **Keyboard-Shortcuts vollstaendig** — Pfeil-hoch (letzte Nachricht bearbeiten), Esc (als gelesen markieren)
- [ ] **Unsend** (Cmd+Z innerhalb 15 Sek.)
- [ ] **Nachricht als Lesezeichen speichern** — Bookmark-Icon im Hover-Menue
- [ ] **Online-Status im DM-Header** — Zeigt ob Gespraechspartner online
- [ ] **Workspace-Name und Icon** in der Sidebar-Kopfzeile
